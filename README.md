# Pomodoro

En Pomodoro-timer med aktivitetsloggning och statistik.

## Funktioner

- **Timer**: 25 minuters arbetspass, 5 minuters vila
- **Aktivitetsloggning**: Logga vad du jobbade med efter varje pass
- **Aktivitetshistorik**: Välj bland tidigare aktiviteter, sorterade efter senaste
- **Statistik**: Dagsöverblick, veckoöverblick och månadsöverblick med tid per aktivitet

## Användning

Öppna `index.html` i en webbläsare. All data sparas lokalt i webbläsarens localStorage.
